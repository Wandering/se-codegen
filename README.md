1. 所有的系统都应该有基本的用户及权限管理,需要初始化生成后这些功能可用;

2. 菜单,功能权限的初始化数据需要可以生成;

3. 集成用户上下文,将用户相关信息集成到页面显示上;

4. 集成基本的操作日志;

5. 国家\省市区功能;

6. GPS 地理位置;
